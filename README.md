# Ibibba

15x5 Ortholinear low profile mechanical keyboard.

![image](docs/PCB_render.png)
![image](docs/PCB_render_back.png)

- USBC
- Bluetooth
- NRF52840
- Low profile
    - Gateron low profile
    - Kailh choc v1 & v2
    - Cherry MX
    - Redragon low profile
- all 1U and 2U
- swiss cheese PCB
- Rotary knob
